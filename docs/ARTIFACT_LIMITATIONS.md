# Artifact Limitations

- Raw market data are not redistributed. The input data are public or publicly
  obtainable, but provider redistribution terms differ.
- The full ten-seed neural rerun can be time-consuming. Cached aggregate
  summaries are included for rapid verification of manuscript tables and
  figures.
- The artifact is designed to support the placement-quality diagnostic in the
  paper. It is not a general-purpose portfolio optimizer or a benchmark suite
  for financial forecasting models.
- The node-exposure mechanism file contains derived test-state scores and
  induced exposures. It is included to regenerate the mechanism figure without
  redistributing raw market returns.
- Small numerical differences can arise across hardware, BLAS libraries, and
  PyTorch versions. The paper tables use cached seed-clustered episode
  summaries generated from the reported ten-seed run.
